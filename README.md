# TabScroller

A Microsoft Edge / Chrome extension: hold right-click + scroll to switch tabs.

## Usage
- Hold right-click anywhere on a page
- Scroll up = previous tab, scroll down = next tab
- Works on all tabs including pinned

## Keyboard shortcuts
- `Ctrl+Shift+Right` — next tab  
- `Ctrl+Shift+Left` — previous tab
